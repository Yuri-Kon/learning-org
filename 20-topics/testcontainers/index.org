#+TITLE: Testcontainers

* Testcontainers 学习索引

** 基础认知

1. Testcontainers 是什么
   - 定义：基于Docker集成测试工具库
   - 核心思想：用真实依赖替代mock
2. 它解决什么问题
   - 本地依赖环境复杂
   - CI 环境不一致
   - mock 不可信
3. 与传统测试方法对比
   - 单元测试(JUnit + Mock)
   - 集成测试(SpringBootTest)
   - Testcontainers定位

** 核心原理

1. 基础架构
   - Java API -> Docker API -> Container
   - 生命周期控制
2. 容器生命周期
   - 启动(start)
   - 等待策略(WaitStrategy)
   - 销毁(stop)
3. 核心抽象
   - ~GenericContainer~
   - ~JdbcDatabaseContainer~
   - ~Network~
   - ~DockerClinet~
4. 与Docker的关系
   - 依赖Docker daemon
   - 不需要手写 docker-compose
