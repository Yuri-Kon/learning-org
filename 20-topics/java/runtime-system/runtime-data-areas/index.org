#+TITLE: JVM 内存结构(Runtime Data Areas)

* 本章目标

- 一个 Java 程序运行时内存如何划分？

- 为什么会出现 StackOverflowError / OutOfMemoryError?
- GC 主要作用于哪一块内存？
